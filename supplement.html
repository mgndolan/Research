Website Source Code

Interested in building your own project website and like this one? Feel free to check out the website github repository and modify this to make it your own. I designed this site in a way that is easy for researchers to replicate who aren't into website development. While this is a very basic website, there are many ways to change it and make it your own. This can simply be the backbone to a more personalized site.

Anyone is welcome and encouraged to contact me with questions on how to make their own research sites like this one.
